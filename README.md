# Portfólio de Mateus Ferreira Félix

Este é um portfólio simples criado com HTML5, CSS3 e JavaScript.

## Estrutura do projeto

- `index.html` - página principal
- `style.css` - estilos do tema escuro e layout responsivo
- `script.js` - renderização dinâmica dos projetos

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Edite `script.js` para adicionar novos projetos ao portfólio.
3. Edite `style.css` para ajustar cores e layout.

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `style.css`, `script.js` e `README.md` para o repositório.
3. No GitHub, vá em `Settings` > `Pages`.
4. Selecione a branch principal (`main` ou `master`) e a pasta `/root`.
5. Salve as configurações.
6. Acesse o site gerado pelo GitHub Pages.

> Exemplo de URL: `https://seu-usuario.github.io/nome-do-repositorio/`
